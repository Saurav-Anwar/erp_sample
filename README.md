# erp_sample

This is a small sample mobile ERP application for a construction company **ABC**.

The app is supposed to be used by **Managers** and **Project leads** to:
1.  Monitor projects
2. Track budgets and expenses
3. Review tasks and team assignments
4. Check payment and approval status

The app is developed by _Saurav Anwar_, using **flutter** `v3.38.5`
